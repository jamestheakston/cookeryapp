Getting Started with Cookery

Welcome to Cookery! This guide will help you set up and start using the Cookery extension to generate delicious recipes.

Step 1: Install Cookery

Download and install the Cookery extension from the Raycast Store. Once installed, you'll need to configure it with your API key.

Step 2: Enter Your API Key

To use Cookery, you'll need an API key. You can get one from the Cookery website. Enter your API key in the settings to activate the extension.

Step 3: Select Your Provider

Choose your preferred AI provider from the dropdown menu. Cookery supports multiple providers for generating recipes.

Step 4: Select Cooking Style

Choose your preferred cooking style from the available options. This helps tailor the recipes to your preferences.

Step 5: Enter Your Prompt

Type what you'd like to cook or the ingredients you have. Be as specific or as general as you like!

Step 6: Generate Your Recipe

Press Enter or click the generate button to create your recipe. Cookery will use AI to generate a detailed recipe based on your prompt.

Step 7: Raycast Integration

Cookery integrates seamlessly with Raycast. You can also access it directly from Raycast for quick recipe generation.
